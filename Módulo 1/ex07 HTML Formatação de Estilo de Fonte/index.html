Principais Formatações

Negrito

Neste texto usaremos a tag "b" para deixar este texto em negrito. Esta tag é uma tag Não Semantica. isto é, tem apenas forma, mas sem sentido

Já este texto, usando a tag strong temos um texto em destaque com Semantica

Itálico

Neste texto, usando a tag "i", temos um texto em Itálico sem Semantica

Já neste, tempos um texto em itálico usando a tag "em" com Semantica

Marca Texto

Podemos tambem criar um texto marcado usando a tag "mark". Mas se voce quiser mudar a cor do marca texto, pode-se usar CSS dentro da tag "mark" usando um parametro chamado "style" e definir o background color. Assim mudaremos a cor para verde ou para azul

Texto Grande e Pequeno

Estamos criando um texto grande com a tag "big" (está obsoleta) e um texto pequeno com a tag "small" neste parágrafo

Texto Deletado

Podemos marcar um texto com excluido para indicar que ele pode ser lido, porem não considerado usando a tag "del"

Texto Inserido

Podemos marcar um texto para indicar que ele foi inserido usandpo a tag "ins"

Existe também o sublinhado, que usamos a tag "u"

Texto Sobrescrito

Serve para inserir coisas do tipo x20 + 3 usando a tag "sup"

Texto Subscrito

Para inserir coisas do tipo H2O usando a tag "sub"
