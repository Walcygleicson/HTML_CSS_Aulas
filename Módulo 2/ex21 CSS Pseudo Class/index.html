Pseudo Classes

Para Que Serve?

Pseudo Classes são identificadas com : (Dois Pontos) nas CSS

Pseudo Classes está relacionado ao estados das coisas, como por exeplo: Queremos que ao passar o mause sobre os quadrados eles mudem de cor.

Como Usar

No Style escrevemos Div: e escolhemos o metodo Hover (significa quando o mause estiver passando por cima)

É Possível Tambem Recolher Textos Dentro de Outros Textos

Vamos recolher um texto "Texto Escondido" dentro de um texto, para isto, usamos a escrevemos dentro de uma div

Para dizer que um elemnto está dentro de outro em CSS usamos ">" desta forma:

div#click > p {}

Está dizendo que o o p está dentro da div. Dentro dos colchetes escrevemos display:none; isto é, os paragrafos que estão dentro da div não será mostrada. Veja que o paragrafo dentro da div sumiu

Para mostrarmos o texto quando o mause for passado sobre o texto, escremos que quando o mause passar sobre o o texto que possui um tetxto filho, ele será exibido. Asim se esceve

div:hover > p { display:block;}
