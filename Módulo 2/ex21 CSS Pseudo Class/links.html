Pseudo Elementos

O Que É?

Pseudo elementos mexem no contúdo do elemento.

Como Usar

Os Pseudo Elementos são identificados por :: (duplo dois pontos)

Um exemplo, vamos colocar um indicador no final de cada link indicando que o conteúdo se trata de um link, para isso usaremos o metodo after e dento dele usaremos o content:"texto";
