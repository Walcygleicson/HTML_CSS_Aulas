GRADIENTE

Para fazer um grandiente em CSS vamos usar o gradiente-image e dentro dele usaremosuma função interna cahamda linear-gradient()

O primeiro parametro é referente á direção do gradinete, e os outros parametros são as quantidades de cores. Ex:
background-image: linear-gradient( to right, white, blue);

Ao usar o gradiente de baixo para cima ou o inverso, o grandiente irá ficar zebrado. Para consertar isso vamos usar uma "Configuração Global e configurar a altura das secções. Ex:
* { height:100%; }

Podemos tamebem usar outra função chamado de radial-gradiente, e no lugar da posição ao angulo do gradiente, usamos o formado.Ex:
background-image: radial-gradient(circle, rgb(160, 23, 160),rgb(146, 18, 129), rgb(155, 21, 54));

Podemos tambem definir a quantidade de ocupação de cada cor no gradiente. Basta escrever uma porcentagem depois de cada cor para definir a ocupação dela
