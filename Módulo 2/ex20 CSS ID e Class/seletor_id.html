SELETORES PERSONALIZADOS

Usando IDs

Como Usar

Como parametro da tag colocamos id="nome da id"
Nas CSS para configurar essa tega escrevemos
Ex:
h1#nome da id {
}

Usamos a # hashtag para definir a id

Usando Class

Como Usar

Não se pode usar dois nomes de id em um mesmo documento. Se eu quero configurar duas tags usando as mesmas configurações eu devo usar o parametro Class que na folha de CSS é reconhecido pelo . (ponto)

Veja no código configurando os h2 escritos "como usar"

Criaremos uma Class como parametro e colocaremos o nome de "comousar"

Usando Duas Class em Um Unico Elemento

Como Usar

É possível tambem usar mais de uma classe em uma tag. Basta vc criar uma class na folha de CSS e dentro de Class="nome" colocar o nome das duas class que vc desjea usar.

Por Exemplo:
Vamos usar o h1 deste tópico com a class "tema" e a class "italico".
NÃO PRECISA SEPARAR OS NOMES DE CLASSE COM VÍRGULA. Veja como fica o código no exemplo abaixo

h1 class=" tema italico"

Nota-se que o h1 recebe o estilo de cores da class tema e o estilo de fonte da class italico
