Como Codar CSS Externamente

  • Estilos Externos significa codar os estilos em um arquivo separado ao arquivo dos códigos HTML
  • Na tag "head" insira a tag "link" e no href digite o nome do seu arqivo css ou crie o arquivo e insira o caminho
  • A tag "link" serve para importar e aplicar os códigos em CSS do outro arquivo. Semelhante ao Import do Python
  • Coloque esta linha de tag em todas as páginas HTML que todas irão adquirir os mesmos estilos
