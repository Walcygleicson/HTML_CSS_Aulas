O Que É?

Os dois tipos de caixas mais usadas são as divs (box-level) e as span (inline-level). O grouping tags consiste em um agrupamento de caixas dentro de caixas

Construindo Este Site

Usaremos caixas semanticas para esta construção. Usaremos a tag header para o cabeçalho do site, a tag main para o conteúdo principal do site e a tag footer para o rodapé. Essas tres tags também são consideradas divs e são box-level.

  • Tags de Contrução Semantica
    • O título deste site ficará na tag de cabeçalho, e é nela que podemos colocar menus de navegação com a tag nav.
    • Todo o conteúdo principl ficará dentro do main e dentro dela podemos dividir todo o conteúdo em seções com a tag section. Separeremos os assuntos deste site em seções. O h2 e seu assunto, assim como o h3 e as ul e seus assuntos ficarão separadas por seções e cada seção será indentificda por uma id intro, assunto e conteudo.
    • Dentro de cada section podemos ter tambem os artigos usando a tag article. Simularemos um artigos abaixo para aplicar a tag. No artigos podemos ter um título de um texto.
    • Dentro de um artigo podemos adicionar uma tag chamada aside que se trata de um conteúdo periférico ao artigo, como por exemplo, uma sitação de quem escrevou tal artigo. Vamos ao exemplo neste h4 Exemplo de Artigo
