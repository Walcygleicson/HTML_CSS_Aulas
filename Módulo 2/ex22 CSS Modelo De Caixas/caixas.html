MODELOS DE CAIXAS

O Que É?

Todo elemento escrito em HTML está dentro de uma caixa, que pode ser configurada nas CSS. Podemos editar esta mesma caixa, que inicialmente fica invisível, usando os estilos.

Podemos criar uma borda ao redor dest caixa usando border

Esta borda ficará bem encostada com as letras dentro da cixa. Para darmos um espaço entre as letras e a borda usamos o padding que cria um preenchimento entre elas, fasendo assim que a borda se distancie do texto

Com a borda já criada, agora temos as possibilidade de adiconar um contorno nesta borda usando o outline. Este contorno permanece grudada na borda.

Se desejarmos criar outra borda ou caixa por fora, esta borda acabrá por ficar colada na borda interna. Então para darmos um espaço entre a borda interna e externa, usaremos o margin que cria uma margem entre a borda e o exterior

Existem Dois Tipo de Box

  • Caixa do tipo BOX-LEVEL

      • Se temos um aparagrafo e criamos um box-level, este sempre será criado abaixo do paragrafo e sempre terá a largura total da tela e o conteúdo ira pular para a proxima linha

    • Um exemplo de Box-Level é o div

      • Criamos uma div sempre que quisermos quebrar a linha

      Outros exemplos são:

      • h1
      • p
      • main
      • video
      • nav
      • ...

  • Caixa do tipo INLINE-LEVEL

      • Se temos um paragrafo e criamos um inline level, este não irá pular para a proxima linha, será criado logo apos o final do paragrafo (se houver espaço), terá a largura exata apenas do conteúdo posto dentro dele, e apos isso o conteudo do paragrafo contininua na mesma linha logo apos a caixa.

    • Um exemplo de Inline-Level é o span

      • Usamos um span sempre que quisermos introduzir uma caixa entre uma linha

      Outros exemplos são:

      • a
      • code
      • input
      • label
      • select
      • ...

Obs Sobre ShortHand:

Quando os quatros valores de uma shorthand forem iguais, como por exemplo "padding:10px 10px 10px 10px;" podemos simplificar mais ainda informando apenas um valor. Assim o computador entende que todos os lados receberão este mesmo valor de 10px. Assim: "padding:10px;" também funciona
