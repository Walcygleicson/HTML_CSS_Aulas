Como Codar CSS

  • Para codar em CSS fora das linhas de html, precisamos criar uma tag "style" (dentro do head) e tudo que for escrito nele será CSS
  • Para selecionar quais tags será editada, basta escrever o nome da tag sem os < > e abrir colchetes
  • Primeiro vamos editar o corpo do site. Então dentro da tag style escrevemos "body {}" e já se pode começar a codar dentro dos colchetes. Tudo ali dentro será referente somende ao body
