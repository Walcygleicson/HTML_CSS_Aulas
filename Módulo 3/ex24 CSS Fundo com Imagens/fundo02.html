Backgroung Com Imagens

background-position

Centralize a imagem usando o background-position: center top;

background-repeat

Para retirar as quebras use background-repeat: no-repeat;

Vamos colocar o background-repeat em repeat-y para a imagem se repetir apenas no eixo y

background-size (contain e cover)

Para resolver o problema do fundo preto aparecendo vamos usar o background-size e dentro dele vamos usar o contain. O contein manda a imagem ser exibida sem alterações. Ou podemos usar o cover no lugar do contain. O cover recorta a imagem para caber 100% na tela.

background-attachment

Mas note que se houver um conteúdo muito grande onde haverá barra de rolagem, ao logar para baixo o fundo escuro voltará a aperecer

Vamos deixar a imagem de fundo travada para que ela não role. Usaremos o background-attachment no fixed Desta forma a imagem ficará fixa mesmo se houver rolagem para baixo ou cima

O resultado que temos aqui é uma conbinação do background-size: cover; com background-attachment:fixed;
