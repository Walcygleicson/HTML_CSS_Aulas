Tabelas Simples

Como Faz?

Criaremos uma tabela usando a tag table

  • Hierarquia de Tabela Simples

  • Table - Tabela
    • Table Row - Linhas de tabela
      • Table Header - Cabeçalho de tabela
      • Table Data - Dados de tabela

Criaremos a primeira linha da tabela, que será um Table Row, usando a tag tr.Em seguida, como mostra a lista de hierarquia, dentro do Table Row temos um Table Data, usando a tag td

Configurar as brodas das células do style com border. As células veem com ar bordas separadas, para unir , no Table, usaremos a função border-collapse com o valor collapse

Criaremos 4 linhas neste exemplo

A1 B1 C1
A2 B2 C2
A3 B3 C3
A4 B4 C4


Alinhamento

Alinhando o Conteúdo das td

O alinhamento horizontal se faz com text-align

O alinhamento vertical se faz com vertical-align

A2 B2 C2

  • Hierarquia de Tabela Grande

  • Table - Tabela
    • thead - Cabeçalho da Tabela
      • tr
      • td
      • th
    • tbody - Corpo da Tabela
      • tr
      • td
      • th
    • tfoot - Pé da Tabela
      • tr
      • td
      • th
