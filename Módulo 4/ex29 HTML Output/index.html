Output

A tag output é usada para mostrar a saída de algo que acontece no html, como por exemplo, uma soma entre dois valores. Para isto, na tag input, usando o parametro oninput é possível fazer este calculo e mostrar no output. O calculo é escrito em JavaScript, desta forma:
soma.innerHTML = Number(num1.value) + Number(num2.value)

Isto é, o html que está dentro do name soma, que neste caso é o output, vai receber e mostrar a soma do valor de cada inout convertido em número.

Obs: O valor do calculo não será enviado, pois o output serve somente para mostrar o calculo.
