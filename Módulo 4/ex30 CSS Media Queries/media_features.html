Media Features

Crie um novo link de estilo css e use o media="all". As configurações feitas deste arquivo valerão para todos os tipos de mídias. Feito isto, cria mais dois arquivos com media screen, destinados a orientação das telas. Um para a tela em retrato, outro para a tela em paisagem. Agora para adiconar as Medias Features, ainda dentro do parametro media, após o screen, deve-se escrever and (orientation: portrait)

Assim: media="screen and (orientation: portrait)"

Esta configuração é para a orientação da tela em modo retrato. Para o modo paisagem escreva landscape.

Assim: media="screen and (orientation: landscape)"
