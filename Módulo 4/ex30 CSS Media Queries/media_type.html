Media Type

Para começar a trabalhar com media query crie um link para um estilo css. Feito isso, nele use o parametro media="screen". Isto é, está indicando que todo o estilo será destinado à tela. Crie outro link para outro estilo css e use o media="print". Este estilo será para inpressora

Feito isto, você já está usando media queries, porém não completamente, pois isto é chamado de media type, arquivos diferentes para meios diferentes. O estilo com media screen será o estilo principal, destinado ao conteúdo mostrado na tela no computador, já o media print será destinado ao conteúdo impresso em papel, logo, as alterações feitas neste aquivo so serão visiveis quando o conteúdo for imprimido. Aperte Ctrl + P para ver a miniatura de impressão.

Media Features

Media type e media features juntas formam as media queries. Enquanto as media types são os tipos de mídias, as tres que está sendo usadas (all, screen e print), as media features são as características desses tipos de mídias.
